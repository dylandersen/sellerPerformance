# Seller Performance

Salesforce Lightning Web Component and Apex classes for analyzing seller performance metrics.

## Components

### Lightning Web Components
- **sellerPerformance** - Main LWC component for displaying seller performance data

### Apex Classes
- **SellerPerformanceAnalysis** - Analysis logic for seller performance
- **SellerPerformanceController** - Controller for the LWC component
- **SellerPerformanceMetrics** - Metrics calculation and aggregation

## Setup

1. Deploy the metadata to your Salesforce org
2. Assign appropriate permissions to users who need access
3. Add the `sellerPerformance` component to your Lightning pages

## Usage

The component displays seller performance metrics and analysis based on configured criteria.
